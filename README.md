# Beat the Press ⚽

A football guessing game where you and an AI try to guess each other's mystery player before time runs out.

## How It Works
- Both sides get a mystery player
- Clues are revealed each round
- Guess your opponent's player before they guess yours
- 90 seconds per leg, two legs per match
